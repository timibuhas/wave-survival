# Wave Survival - 3D Shooter Game

Wave Zombie Survival este un joc de tip wave-based shooter dezvoltat în Unity, realizat ca proiect universitar. Jucătorul trebuie să supraviețuiască valurilor de zombi din ce în ce mai dificile, fiecare rundă crescând numărul și dificultatea inamicilor.

Proiectul demonstrează implementarea mecanicilor de bază ale unui joc shooter, inclusiv spawning-ul inamicilor, sistemul de combat și progresia dificultății.

---

## Tehnologii utilizate
- **Game Engine:** Unity  
- **Limbaj de programare:** C#  
- **Motor fizic:** Unity Physics  
- **UI System:** Unity UI  

---

## Funcționalități principale
- Sistem de wave-uri cu dificultate progresivă  
- Spawn automat al zombilor la fiecare rundă  
- Control al jucătorului (mișcare și tragere)  
- Sistem de combat și eliminare a inamicilor  
- Sistem de supraviețuire bazat pe valuri  
- Interfață UI pentru afișarea informațiilor din joc  
- Sistem de Game Over și restart  

---

## Ce am învățat
- Dezvoltarea jocurilor folosind Unity și C#  
- Implementarea unui sistem de spawning pentru inamici  
- Crearea unui sistem de progresie a dificultății  
- Gestionarea coliziunilor și fizicii în Unity  
- Organizarea logicii jocului folosind principii OOP  
- Crearea și gestionarea interfeței utilizator
